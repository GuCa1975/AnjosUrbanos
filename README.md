# Anjos Urbanos Virtual 💇‍♀️✨

**Salão Virtual com Inteligência Artificial** — Experimenta novos cortes e cores de cabelo antes de os fazer, usando o poder da IA Gemini.

## Funcionalidades

- 📸 **Captura de Foto** — Tira uma selfie ou carrega uma foto existente
- 🎨 **Catálogo de Estilos** — Presets de cortes e cores (Loiro Platinado, Ruivo, Rosa Pastel, etc.)
- 🔄 **Comparador Antes/Depois** — Slider interativo para ver a transformação
- 💎 **Design Dark Luxury** — Interface sofisticada com tema preto e dourado
- 🤖 **IA Gemini** — Transformações fotorrealistas preservando a identidade facial

## Como Executar Localmente

1. Clona o repositório:
   ```bash
   git clone https://github.com/GuCa1975/AnjosUrbanos.git
   cd AnjosUrbanos
   ```

2. Instala as dependências:
   ```bash
   npm install
   ```

3. Cria o ficheiro `.env` com a tua chave de API:
   ```bash
   cp .env.example .env
   # Edita o .env e adiciona a tua VITE_GEMINI_API_KEY
   ```

4. Obtém a tua chave de API em: [aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey)

5. Inicia o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

## Deploy no Replit

1. Importa este repositório no [Replit](https://replit.com)
2. Adiciona `VITE_GEMINI_API_KEY` nos **Secrets** do Replit
3. Clica em **Run** e depois em **Deploy**
